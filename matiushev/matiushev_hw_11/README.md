# 📌 Simple Site — SCSS Refactoring

## 📖 Опис виконаної роботи

У цій роботі було виконано рефакторинг існуючого CSS-коду шаблону **Simple Site** з переходом на SCSS (Sass).
Основна мета — покращити структуру стилів, підвищити читабельність коду та зробити його більш масштабованим.

## 🚀 Виконані завдання

### 🔹 1. Декомпозиція стилів

CSS було розбито на логічні модулі відповідно до архітектури SCSS:

* `abstracts` — змінні та міксіни
* `base` — базові стилі та reset
* `components` — перевикористовувані елементи (кнопки)
* `layout` — стилі окремих секцій сайту

---

### 🔹 2. Перехід на SCSS

* Замінено всі CSS-значення на **SCSS-змінні**
* Впроваджено вкладеність селекторів (nested syntax)
* Оптимізовано повторювані стилі

---

### 🔹 3. Використання міксинів

Створено універсальні міксіни для:

* Flex-центрування
* Контейнера
* Адаптивності (media queries)

---

### 🔹 4. Адаптивність через SCSS

Замість звичайних `@media` використано міксіни:

```scss
@include tablet { ... }
@include desktop { ... }
@include phone { ... }
```

---

### 🔹 5. Оптимізація структури

* Покращено читабельність коду
* Зменшено дублювання
* Дотримано методології БЕМ
