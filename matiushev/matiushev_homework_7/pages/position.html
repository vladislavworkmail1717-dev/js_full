Робота з Position

Інтерактивний довідник з CSS позиціонування

📖 Основи

Властивість position визначає систему координат для елемента.

🧪 Практична демонстрація

Static: Звичайний порядок у документі.
Static
Relative: Рухається відносно свого початкового місця, але зберігає простір за собою.
Relative (top:15, left:25)
Absolute: Виривається з потоку. Позиціонується відносно найближчого батька з position не-static.
Absolute
Батьківський блок (relative)
Sticky: Гібрид relative та fixed. Залишається у видимій зоні контейнера при скролі.
