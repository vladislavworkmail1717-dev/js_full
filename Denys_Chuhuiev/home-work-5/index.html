Пройдений матеріал (основи Git)

Під час уроку я вивчив базовий Git-процес: від ініціалізації репозиторія до фіксації змін. Це дає змогу контролювати версії проєкту та безпечно зберігати прогрес.

  1. Створення репозиторію: git init
  2. Перевірка стану: git status
  3. Додавання змін: git add
  4. Фіксація змін: git commit
  5. Перегляд історії: git log
  6. Відправка на віддалений репозиторій: git push
