# Домашнє завдання №5 — Git базовий робочий процес

## Проєкт

Даний проєкт — простий двосторінковий сайт на **HTML/CSS**, що описує виконання домашнього завдання №5 з Git:

- `index.html` — головна сторінка (огляд того, що зроблено).
- `hw5.html` — детальна сторінка з поетапним описом матеріалу.
- `style.css` — спільний стиль для обох сторінок.

## Функції

- адаптивна верстка (працює на мобільних і десктопних екранах;
- навігація між двома сторінками;
- структурований опис пройденого матеріалу (`Git`).

## Що зроблено (план робочого процесу)

1. Ініціалізовано репозиторій.
2. Додано базову HTML/CSS структуру.
3. Оформлено зміст сайту.
4. Підготовлено детальний звіт про пройдені команди Git.
5. Описано особливості проєкту в цьому README.

## Команди Git, які використано

- `git init`
- `git add`
- `git commit`
- `git remote add origin <URL>`
- `git push -u origin master`
- `git status`
- `git log`

## Як здати завдання

У відповіді на ДЗ надсилаються два посилання:

- посилання на репозиторій: `https://github.com/<your-username>/home-work-5`
- посилання на GitHub Pages: `https://<your-username>.github.io/home-work-5/`

> Перед відправкою перевірте, що сторінка доступна за посиланням GitHub Pages і правильно відображає
> `index.html` і `hw5.html`.
