# Домашні завдання FullStack iTHillel

Цей репозиторій містить мої домашні роботи з курсу **FullStack** від школи **iTHillel**.

## Що знаходиться в репозиторії

Кожне домашнє завдання розміщене в окремій папці. Головна навігація винесена на сторінку `index.html`, з якої можна перейти до кожної роботи.

## Домашнє завдання №13

Тема роботи: **SVG, A11Y, DevTools, Project optimization**.

Що реалізовано:

- створено SVG-полотно з квадратом, прямокутником, колом та еліпсом;
- для кожної фігури задано власні кольори заливки та контуру товщиною `5px`;
- для еліпса додано CSS-анімацію зі зміною кольору заливки та обводки;
- додано мінімальний адаптивний дизайн і базову доступність через `title`, `desc` та семантичну структуру;
- на сторінці розміщено посилання на GitHub Pages і на папку з кодом у репозиторії.

## Як переглянути

1. Відкрийте головний файл `index.html`.
2. Оберіть потрібне домашнє завдання в списку.
3. Перейдіть за кнопкою на сторінку відповідної роботи.
