Функції JavaScript на основі шаблону з уроку

На цій сторінці зібрані три реалізовані функції: привітання за ім'ям, генерація випадкового цілого числа в діапазоні та підсумовування великих чисел через BigInt.

Шаблон: base (1) 3 реалізовані функції HTML + CSS + JavaScript

Що реалізовано

Усі завдання з шаблону-заготовки виконані у JavaScript. Нижче відображаються короткі описи, приклади запуску та результати виконання кожної функції.

1. `greetByName(msg, name)`

Функція приймає текст привітання та ім'я, після чого повертає готовий рядок у форматі Hi, John.

2. `getRandomInt(min, max)`

Функція генерує випадкове ціле число в діапазоні від min до max включно за допомогою Math.random() і Math.floor().

3. `sumBigIntegers(numStr1, numStr2)`

Функція перетворює рядки з великими числами у BigInt та повертає їх суму без втрати точності.

Домашнє завдання №19 оформлене як окрема сторінка з прикладами запуску функцій і результатами на екрані.
