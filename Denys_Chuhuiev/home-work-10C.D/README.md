# Home Work 10

Responsive version of the "Simple Site Template" based on homework 9.

## Description

This project adapts the existing landing page for different screen sizes starting from 375px.
The page was built with:

- HTML5
- CSS3
- CSS media queries
- Fluid spacing with flexible units
- Flexbox
- JavaScript

## Features

- semantic page structure
- responsive layout for mobile, tablet, and desktop screens
- flexible container widths and adaptive spacing
- statistics section that rearranges at smaller breakpoints
- video block with animated play button and local `.mp4` file
