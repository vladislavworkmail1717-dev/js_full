Багатосторінковий сайт кав'ярні з єдиним стилем оформлення

Цей сайт створений як навчальний проєкт для демонстрації візуального оформлення багатосторінкового веб-сайту. На сторінках використано спільний контейнер, однакове меню, один стиль для футера, окрему галерею та охайну форму.

Уся стилізація побудована через класи, а над основними HTML-блоками додані коментарі з описом логіки розміщення елементів.

Що перевіряє це ДЗ

  • Наявність контентного контейнера з відступами та центруванням.
  • Однаковий дизайн меню та футера на всіх сторінках.
  • Логічно вирівняну форму без притискання полів до країв.
  • Галерею із зображеннями, розташованими по три в ряд.

Структура навчального сайту

Кожна сторінка виконує окрему роль, але візуально залишається частиною одного цілого завдяки єдиній стилістиці.

01

Інформаційні сторінки

Галерея побудована сіткою по три елементи в рядок для впорядкованого перегляду.
