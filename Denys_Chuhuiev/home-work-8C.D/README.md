# Домашня робота №8

Ця домашня робота виконана за матеріалами уроку №8:
`BEM`, `Float Layout`, `Flexbox`, `Grid Layout`, `Semantic tags`.

## Що реалізовано

- Створено головну сторінку `index.html`
- Створено навігацію між усіма сторінками
- Використано семантичні теги HTML5
- Використано BEM-іменування класів
- Додано коментарі українською мовою перед кожним логічним блоком
- Практичні приклади макетів виконано через `float` і `margin`

## Структура проєкту

- `index.html` - головна сторінка роботи
- `page-1.html` - один блок ліворуч, другий праворуч
- `page-2.html` - один блок ліворуч, кілька блоків праворуч
- `page-3.html` - два блоки ліворуч, два праворуч
- `page-4.html` - три блоки праворуч, два з них один під одним
- `page-5.html` - три блоки по центру, два з них один під одним
- `styles.css` - спільні стилі для всіх сторінок

## Як відкрити

1. Відкрити файл `index.html` у браузері.
2. Перейти на потрібну сторінку через меню навігації.

## Примітка

Для демонстраційних макетів у завданні не використовувалися `flex` і `grid`,
оскільки за умовою домашньої роботи потрібно було застосувати саме `float` та `margin`.
