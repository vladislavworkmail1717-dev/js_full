# Home Work 9

Simple Site for homework 9 from the Full Stack course.

## Description

This project is a responsive landing page created from a Figma layout.
The page was built with:

- HTML5
- CSS3
- CSS variables
- Flexbox
- JavaScript

## Features

- semantic page structure
- responsive layout
- reusable CSS variables for colors and fonts
- statistics section with SVG icons
- video block with play button and local `.mp4` file
