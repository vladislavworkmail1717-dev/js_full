Events, addEventListener та делегування подій

На сторінці виконано задачі з шаблону main.js: обробка кліку по кнопці, відстеження координат миші та делегування кліків для пунктів списку.

Events addEventListener removeEventListener Event object Bubbling Delegation

Що взято з уроку

Урок №26 пояснює типи подій, способи призначення обробників, об'єкт Event, методи addEventListener() і removeEventListener(), а також фази поширення події: захоплення, цільову фазу та спливання.

Event object

Об'єкт події містить тип події, цільовий елемент, координати миші та методи керування стандартною дією або поширенням.

Propagation

Подія проходить фази захоплення, цільову фазу і спливання, а stopPropagation() може зупинити шлях.

Delegation

Один обробник на батьківському елементі може керувати кліками багатьох дочірніх елементів, навіть якщо вони додані пізніше.

1. handleButtonClick()

Функція знаходить кнопку за ID і призначає їй обробник події click, який виводить повідомлення в консоль.

    2. trackMousePosition()

    Функція додає обробник mousemove на document і виводить координати курсору.

    Поточні координати X: 0, Y: 0

    3. setupEventDelegation()

    Функція ставить один обробник на весь список і реагує лише тоді, коли клік відбувся по елементу li.
