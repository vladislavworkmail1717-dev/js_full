# Домашнє завдання №6 — таблиця у стилі Піта Мондріана

## Що зроблено

Цей проєкт — виконання завдання з CSS/HTML:
- відтворити композицію, натхненну роботами Пієта Мондріана;
- використати **лише таблицю** як основу макета;
- дотриматись принципів:
  - мінімальна кількість рядків і стовпців;
  - **без прихованих** рядків/стовпців;
  - об’єднання комірок через `rowspan`/`colspan`;
  - чіткі прямокутні сегменти та кольорові блоки.

## Структура файлів

- `index.html` — фінальна сторінка з виконаним макетом.
- `README.md` — опис домашнього завдання.

## Результат

У макеті:
- побудована сітка таблиці з 6 колонок і 5 рядків;
- задані пропорції блоків;
- застосовані базові кольори Мондріана (жовтий, червоний, синій, білий);
- чорні лінії-межі як частина стилю;
- акцент на чистій геометричній структурі.

## Запуск локально

Відкрийте `index.html` у браузері (достатньо подвійного клацання або через Live Server).
