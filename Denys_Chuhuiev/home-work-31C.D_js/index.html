Promise

HTTP-методи та CRUD

Метод CRUD Призначення
GET Read Отримати ресурс або список ресурсів.
POST Create Створити новий ресурс на сервері.
PUT Update Повністю оновити існуючий ресурс.
PATCH Update Частково змінити існуючий ресурс.
DELETE Delete Видалити ресурс за ідентифікатором.
