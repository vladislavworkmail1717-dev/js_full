# Home Work 7 — Практичні сторінки (Hillel LMS, урок 7)

Цей репозиторій містить виконання домашнього завдання №7:
- бази верстки на `<div>` та `<span>`
- експерименти з `normalize.css` і `reset.css`
- демонстрації `display`
- `box-sizing`
- `float` + `clear`
- `overflow`
- позиціонування (`static`, `relative`, `absolute`, `fixed`, `sticky`)

## Структура

- `index.html` — головна сторінка з навігацією
- `01-basic-div-span.html`
- `02-normalize.html`
- `03-reset.html`
- `04-display.html`
- `05-box-sizing.html`
- `06-float-clear.html`
- `07-overflow-position.html`
- `normalize.css`
- `reset.css`

## Запуск

1. Відкрийте `index.html` у браузері.
2. Перейдіть між сторінками через меню в кожній сторінці.
