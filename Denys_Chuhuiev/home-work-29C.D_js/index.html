Jest: юніт-тести для JavaScript-функцій

У цій роботі налаштовано npm-проєкт із Jest, підготовлено функції ageClassification та weekFn, а також написано тести для всіх значень із прикладів уроку.

Jest Unit Tests Matchers BDD npm test

Що взято з уроку

На занятті розглядали Jasmine і Jest як інструменти для перевірки JavaScript-коду, структуру тестів через describe, test / it та expect, а також підхід BDD, де тест описує очікувану поведінку функції.

Project setup

Налаштування проєкту

  1. Створено package.json командою npm init -y.
  2. Встановлено Jest як dev dependency: npm install --save-dev jest.
  3. У скриптах додано запуск тестів: "test": "jest".
  4. Перевірка виконується командою npm test.
