# 🎨 SVG Shapes Animation Project
Цей навчальний проєкт демонструє створення базових геометричних фігур за допомогою SVG (Scalable Vector Graphics) та їх стилізацію за допомогою CSS3.

# 📋 Опис проєкту
Проєкт містить чотири основні фігури: 
- квадрат, 
- прямокутник, 
- коло 
- еліпс. 

Кожна фігура має унікальні параметри контуру та кольору. 
Головною особливістю є інтерактивна анімація, яка активується при наведенні курсору.

# ✨ Функціонал

## SVG-графіка: 
Використання тегів *rect*, *circle* та *ellipse*.

## CSS Анімація: 
При наведенні (:hover) на будь-яку фігуру запускається анімація @keyframes colorShift.
Плавні переходи: Циклічна зміна кольору заливки (fill) та контуру (stroke) протягом 1 секунди.     

# 🚀 Як запустити
1. Створіть папку для проєкту.
2. Скопіюйте HTML-код у файл index.html.
3. Створіть папку css та збережіть у ній файл style.css з відповідним кодом.
4. Відкрийте index.html у будь-якому сучасному браузері (Chrome, Firefox, Edge).

# ⚙️ Технічні деталі анімації
Анімація colorShift змінює кольори у трьох фазах:
- 0%: Початковий синій колір.
- 50%: Перехід у червоний/фіолетовий спектр.
- 100%: Завершення на жовтому/помаранчевому кольорі.
Використання infinite alternate дозволяє анімації програватися безперервно та плавно в обох напрямках.
